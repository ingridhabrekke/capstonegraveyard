# Welcome to The Capstone Graveyard 

## Tailwind CSS
This project uses Tailwind CSS. To use this framework, make sure Node.js is downloaded.
Run `npm install` to install packages from package.json.

This is only necessary if working on the project. It is not necessary if just viewing the project. 


## Run Application

To run the app, run: 

    $ export FLASK_ENV=development
    $ export FLASK_APP=web
    $ python3 -m flask run
in the web folder.
